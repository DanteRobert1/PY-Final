Datos Disponibles

Los datos disponibles incluyen información detallada sobre los videojuegos y sus ventas, distribuidos en las siguientes categorías:

  • Name: Nombre del videojuego. INT
  • Platform: Plataforma de juego (ejemplo: PlayStation, Xbox, PC). STRING
  • Year: Año de lanzamiento. INT
  • Genre: Género del videojuego (ejemplo: Acción, Aventura, Deportes). STRING
  • Publisher: Editor o distribuidor del videojuego. STRING
  • Ventas por región: INT
    • NA_Sales: Ventas en América del Norte.
    • EU_Sales: Ventas en Europa.
    • JP_Sales: Ventas en Japón.
    • Other_Sales: Ventas en otras regiones.
  • Global_Sales: Ventas globales totales. INT
